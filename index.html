我从一线 Java 开发成长为资深架构师，在交易、支付、风控、清结算等金融核心领域做了 11 年—— 先后在嘉联支付、顺丰科技、华锐金融、菜鸟网络工作，目前在滴灌通担任资深 Java 专家 / 技术负责人。

在滴灌通，我主导了交易所 2.0 系统的架构设计与实现，完成 SPV / SPAC / ETF 等多类金融产品落地， 负责账户体系与清算核心系统重构，支撑平台亿级资产规模、线上零故障。

近两年我把多年系统工程经验投入 AI 方向：用 LangGraph 构建智能投融资分析， 并主导建设 AIWS 多智能体协作平台（AgentScope + A2A 协议）， 把大模型能力真正落到企业的真实工作流里——可理解、可确认、可执行。
