## 设计说明

- **风格**：深石板蓝底（非纯黑，避免压抑）+ 单一克制的钢蓝强调色；高级感主要来自留白、对齐、字重节奏与层次，而非炫酷特效。
- **统一体系**：圆角 / 描边 / 背景层次 / 卡片 / 按钮 / hover 全部走同一套 token，组件风格一致。
- **微交互克制**：仅保留 hover 微位移、reveal 淡入、导航高亮、弹窗开合，不打扰阅读。
- **响应式**：桌面舒展、平板平滑过渡、手机重新排版（卡片堆叠、时间线竖排、导航收起为汉堡菜单），非简单缩放。
- **可访问性**：语义标签、`aria-*`、可见焦点样式、深色下足够对比度。
